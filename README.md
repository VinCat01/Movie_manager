# Каталог фильмов
Простое консольное приложение для управления коллекцией фильмов.

## Описание проекта
Приложение позволяет:

Просматривать список фильмов

Добавлять новые фильмы

Отмечать фильмы как просмотренные

Искать фильмы по году выпуска

Данные хранятся в формате JSON в файле movies.json.

## Установка и запуск 


Клонируйте репозиторий или скачайте файлы проекта

Убедитесь, что у вас установлен Python 3.6+

Запустите приложение:

python app.py



## Требования


Python 3.6 или выше

Нет внешних зависимостей (используются только стандартные библиотеки)



## Особенности реализации


Данные автоматически сохраняются после каждого изменения

Поддержка UTF-8 кодировки

Обработка ошибок ввода пользователя

Чистая архитектура: логика отделена от интерфейса

Иммутабельные функции: возвращают новые списки вместо изменения оригиналов

Проект разработан в учебных целях для закрепления навыков:
